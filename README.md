# RISC-V-CPU-FPGA-Project

Disclaimer:\
This project is developed for educational purposes only. Its primary goal is to provide a hands-on understanding of the RISC-V ISA and CPU design by building a simple RV32I processor from scratch.  

The implementation prioritizes readability and incremental learning over performance, optimization, or compliance with production-quality design practices. As such, it may omit advanced architectural features, optimizations, and verification methodologies that would be expected in a commercial or research-grade processor
